# be-chess

2024 우아한 테크캠프 7기 체스 프로젝트

# how to run
- jdk 17 available
```
cd /java-chess
./gradlew build
./gradlew run
```
# 지원 가능한 기능
- 기본적인 GUI
- 본인의 턴이 아닌 경우 움직이려 할 때, alert
- 이동할 수 없는 경로로 움직이려 할 때, alert
- pawn의 promotion
- 게임 종료
